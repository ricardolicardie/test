Project Overview

Casa del Redentor required a modern, accessible website that would reflect their religious identity while providing key information (events, donations, and spiritual resources). The main challenge was balancing visual appeal with intuitive functionality within Wix's technical constraints.

Project Details

Client
Casa del redentor
Industry
Non-Profit
Platform
Website
Tools Used
Figma, Wix, theme customization

Design Solution

The Casa del Redentor website achieves a sacred yet accessible digital presence through a harmonious blend of spiritual aesthetics and modern functionality. A serene color palette of deep blues (#2E3A59) and warm accents (#E8C547) reflects the organization’s values, while clean typography ("Open Sans" for body text, "Playfair Display" for headings) ensures legibility across all ages. The mobile-first interface features intuitive navigation with a sticky header and touch-friendly buttons, optimized for elderly users and younger congregants alike. Accessibility was core to the design, with WCAG-compliant contrast ratios, scalable text, and ARIA labels for screen readers. Strategic schema markup enhances local search visibility for church services and community events.

Key Features

  • Spiritual Event Hub: Visually-driven presentation with intuitive categories and high-resolution imagery that showcases spiritual events and community activities.
  • Donation Portal: Secure Wix Forms integration with customizable donation tiers and recurring payment options.
  • Fully Responsive Experience: Collapsible menu, optimized media queries, and 1.5s average load time on mobile (via WebP images).
  • SEO-Optimized Content: Localized schema markup for service times and location, boosting "church near me" search rankings by 25%.
